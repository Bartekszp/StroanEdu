Lekcja 9. Metody

Metody są to takie elementy kodu, które gromadzą jakąś funkcjonalność – mogą przyjmować jakieś wartości (parametry), wykonują na nich dowolne operacje, a następnie zwracają wynik (lub nie).

Istnieją trzy główne rodzaje metod:

  1. metody nie przyjmujące parametrów i nie zwracające wartości
  2. metody przyjmujące parametry, ale niczego nie zwracające
  3. metody przyjmujące parametry i zwracające jakiś wynik do programu

Definicja metody:

Definicja metody składa się z kilku elementów (elementy ujęte w nawiasy kwadratowe są opcjonalne):

[Modyfikatory] Typ Nazwa ([Lista argumentów])
{
    [Ciało metody]
}

Elementy definicji metody:

Deklaracja metody:

  • Modyfikatory - określają dostępność metody. Główne modyfikatory dostępu to: public i private. Modyfikator public udostępnia daną metodę na zewnątrz dla innych klas. Modyfikator private umożliwia użycie metody tylko wewnątrz klasy.
  • Typ – typ danej zwracanej przez metodę. Każda metoda może zwracać jakąś wartość i w deklaracji należy podać typ tej wartości. Jeśli nic nie zwraca, to zamiast nazwy typu umieszcza się słowo kluczowe void.
  • Nazwa – nazwa metody, różna od nazwy klasy, w której została zdefiniowana,
  • Lista argumentów – lista argumentów przekazywanych do metody. Lista argumentów zawiera argumenty metody oddzielane przecinkami. Dla każdego argumentu musi być podany typ oraz jego nazwa. Metoda może nie mieć żadnych argumentów, wówczas umieszcza się pustą parę nawiasów.
  • Ciało metody – inaczej treść metody – to kod ujęty w klamrowe nawiasy, który realizuje zadanie przypisane dla metody. W ciele metody można deklarować zmienne lokalne, umieszczać instrukcje i wywołania innych metod. Zadeklarowane zmienne nie mogą mieć tej samej nazwy, co nazwy argumentów. Jeżeli metoda ma określony typ zwracanej wartości (inny niż void), w ciele metody musi być instrukcja return, która zwraca daną określonego typu.

  1. Metoda nieprzyjmująca parametrów i nie zwracająca wartości:

    Przykład:

    static void Main(string[] args)
{
    WypiszImie();
}

private static void WypiszImie()
{
    Console.WriteLine("Moje imie to Marek");
}

  2. Metoda przyjmująca parametry, ale niczego nie zwracająca:

    Przykład:

    static void Main(string[] args)
{
    DodajDwieLiczby(5,10)
}

private static void DodajDwieLiczby(int liczba1, int liczba2)
{
    int wynik = liczba1 + liczba2;
    Console.WriteLine("Wynik dodawania to {0}",wynik)
}

  3. Metoda przyjmująca parametry i zwracająca wynik działania do programu

    Przykład:

    static void Main(string[] args)
{
    float wynikDzieleniaMetody = OdejmijLiczby(5, 10);
}

private static float OdejmijLiczby(float liczba1, float liczba2)
{
    return liczba2 - liczba1;
}
