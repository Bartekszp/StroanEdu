Lekcja 6. Switch

Podobną rolę jak instrukcja If pełni inna instrukcja warunkowa - Switch. Sprawdza się w sytuacji, gdy program komputerowy powinien w odpowiedni sposób reagować na proste warunki w postaci pojedynczej wartości zmiennej. W zależności od wartości tej zmiennej, wykona się inna część programu. Warto wtedy użyć instrukcji wielowarunkowej switch..case. Początkowo składnia polecenia switch..case może się wydać skomplikowana. Jednak po jej przeanalizowaniu, okazuje się, że jej budowa jest przejrzysta i uporządkowana, a sposób użycia jest intuicyjny.

Składnia polecenia Switch:

switch (wyrażenie)
{
case wartość wyrażenia:
    polecenie lub polecenia;
    break;
case druga możliwość:
    polecenie lub polecenia;
    break;
default:
    polecenie lub polecenia;
    break; //opcjonalnie}
  • switch - nazwa instrukcji
  • wyrażenie - wyrażenie którego wartość będzie sprawdzana
  • case wartość zmiennej/wyrażenia - wejście w instrukcję następuje gdy wartość zmiennej zgadza się z wyrażeniem(odpowiednik if). Należy pilnować żeby dwa przypadki case nie sprawdzały tej samej wartości, ponieważ jest to błąd.
  • break - słowo kończące instrukcje, jeżeli warunek został wykonany to nie ma potrzeby sprawdzać kolejnych warunków
  • default - instrukcja domyślna, czyli taka która wykona się jeżeli nie wydarzy się żaden wcześniejszy warunek (odpowiednik else). break i default są opcjonalne.
static void Main(string[] args)
{
    Console.WriteLine("Jaką ocenę dostałeś?");
    char grade = char.Parse(Console.ReadLine());
    switch (grade)
    {
        case '5':
        case '4':
            Console.WriteLine("Bardzo dobrze");
            break;
        case '3':
            Console.WriteLine("Nieźle");
            break;
        case '2':
            Console.WriteLine("Lepiej to popraw!");
            break;
        case '1':
            Console.WriteLine("Nie zdasz!");
            break;
        default:
            Console.WriteLine("Nieprawidłowa ocena");
            break;
    }
}
