Lekcja 4. Operatory matematyczne i logiczne.

Operatory

Operatory matematyczne to nic innego jak elementy znane doskonale z matematyki, pozwalające na porównywanie dwóch wartości. Do znanych operatorów zaliczamy:

  • znak większości (>),
  • mniejszości (<),
  • większe lub równe (>=),
  • mniejsze lub równe (<=),
  • równe (==) oraz różne (!=).

Oraz operatory logiczne:

  • Logiczne „i” (&&) (wszystkie warunki muszą być spełnione)
  • Logiczne „lub” (||) (jeden z warunków musi być spełniony)
  • Negacja (wykrzyknik) !true = false; (zmienia wartość logiczną na przeciwną)

Przy ich pomocy możemy zbudować proste i złożone zdania logiczne. Warto dodać, że takie zdania mają wartość logiczną prawda/fałsz (true/false). Dzięki temu możemy przypisać je do jednego z poznanych typów zmiennych.

Oczywiście chodzi o typ bool.

Przykład:

bool wynik = 5 > 3;
Console.WriteLine( wynik );
wynik = "ala" == "ma kota";
Console.WriteLine( wynik );

Należy zwrócić uwagę na znak dla operatora równości == i uważać, aby nie mylić go z operatorem przypisania =. Dwie instrukcje x = y oraz x == y to dwie różne operacje. Pierwsza z nim przypisuje wartość y do x, natomiast druga porównuje ich wartość.

Aby możliwe było porównywanie dwóch wartości, muszą być one tego samego typu. Nie można porównać np. zmiennej typu string „Ala” ze zmienną typu int 5. Jest to nielogiczne.

Przykład:

int a = 5;
int b =7;
bool czyRowne = a == b; //false
bool czyRozne = a != b; //true

bool czyWieksza = a > b; //false
bool czyMniejsza = a < b; // true

Operator trójargumentowy

Możemy użyć operatora trójargumentowego żeby zamiast wyświetlania wartości True / False jako wynik, wyświetlić dowolny tekst.

Operator ma następującą składnię:

[wyrażenie_logiczne] ? [wykonaj to, gdy wyrażenie jest prawdziwe] : [wykonaj to, gdy nieprawdziwe]

Przykład:

MessageBox.Show($"Czy trójkąt jest prostokątny? {(czyProstokatny ? "Tak" :
"Nie")} ", "Wynik", MessageBoxButtons.OK);

Inkrementacja i dekrementacja.

Inkrementacja (czyli zwiększanie) lub dekrementacja (zmniejszanie) wartości o jeden jest częstym zadaniem w programowaniu.

W standardowy sposób możemy to zrobić tak:

int liczba = liczba + 1; // do naszej liczby przypisujemy jawnie 1

Możemy ten zapis uprościć:

Int liczba += 1; // przy pomocy operatora += dodajemy nową wartość do naszej zmiennej pomijając odwołanie się do niej po prawej stronie zapisu.

C # ma dwa operatory do tego typu działań, które upraszczają jeszcze bardziej ten zapis. Są to ++ i --.

Operatorów inkrementacji i dekrementacji używa się tylko na typach liczbowych, czyli int, float, double itd. Wykorzystuje się je głównie w pętlach, który poznamy za kilka zajęć. Składnia:

[zmienna_typu_liczbowego]++ //inkrementuje o 1
[zmienna_typu_liczbowego]-- //dekrementuje o 1

Przykład:

int liczba =5;
liczba++; //liczba wynosi teraz 6
liczba--; //liczba wynosci teraz 5;


Uwaga

Operator inkrementacji i dekrementacji może występować przed lub po zmiennej. Gdy występuje przed zmienną, zmiana jej wartości nastąpi w danej instrukcji. Jeżeli występuje po zmiennej, zmiana wartość wystąpi dopiero w kolejnej instrukcji.
